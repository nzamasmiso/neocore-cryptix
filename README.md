# neocore-cryptix

**What this is**: A research lab where I study how systems and attackers behave, then build tooling to detect it. The stack changes with the problem.

**Who it’s for**: SOC analysts, Detection Engineers, AppSec engineers, and builders creating practical security tools.

**What you’ll get**: Full cycle work from research to validated detection. Analyze techniques, prototype logic, test against data, tune for coverage + low noise + analyst trust. Every detection ties to MITRE ATT&CK.

**Goal**: Help defenders detect faster with tooling they can trust.
